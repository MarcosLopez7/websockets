asyncio es una biblioteca para escribir código concurrente usando sintaxis de async/await

import asyncio

async def main():
print('Hello ...')
await asyncio.sleep(1)
print('... World!')

# Python 3.7+
asyncio.run(main())

Asyncio es usado para múltiples frameworks asíncronos de Python que de redes y servicios web, conexión a bases de datos, encolar tareas de forma distribuida, entre otros

API's de alto nivel:

  • Corutinas
  • Network IO and IPC;
  • Controlar subprocesos
  • Distribuir tareas via a colas
  • Sincronizar código concurrente

Corutinas

Son funciones que crean tareas (funciones asíncronas) que corren de manera paralela
